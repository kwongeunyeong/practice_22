# 📝 To-Do 앱

간단하고 아름다운 할 일 관리 웹 애플리케이션입니다. Cursor 내장 브라우저에서 바로 볼 수 있습니다.

## ✨ 기능

- ✅ 할 일 추가 및 삭제
- ✅ 할 일 완료 체크
- ✅ 필터링 (전체 / 진행중 / 완료)
- ✅ 완료된 항목 일괄 삭제
- ✅ 로컬 스토리지를 통한 데이터 저장
- ✅ 반응형 디자인 (모바일 지원)
- ✅ 모던하고 아름다운 UI

## 🚀 Cursor 내에서 사용 방법

1. 터미널에서 `python server.py` 실행
2. Cursor 하단 상태바에 포트 8000이 표시됨
3. 포트 번호를 클릭하면 Cursor 내장 브라우저에서 자동으로 열림
4. 또는 `Ctrl+Click` (또는 `Cmd+Click`)로 `http://localhost:8000` 링크 클릭

## 📖 사용법

1. 할 일을 입력하고 "추가" 버튼을 클릭하거나 Enter 키를 누르기
2. 체크박스를 클릭하여 할 일 완료 표시
3. 필터 버튼을 사용하여 원하는 항목만 보기
4. 완료된 항목은 "완료된 항목 삭제" 버튼으로 일괄 삭제 가능

## 💾 데이터 저장

모든 할 일 데이터는 브라우저의 로컬 스토리지에 저장되므로, 페이지를 새로고침해도 데이터가 유지됩니다.
